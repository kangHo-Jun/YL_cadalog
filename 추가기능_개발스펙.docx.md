**대리점 웹 카탈로그**

**추가 기능 개발 스펙 문서**

v1.0  |  2026-03-04  |  Zart

기존 PDF 뷰어 카탈로그에 아래 2가지 기능을 추가 개발 요청드립니다.

| 기능 | 목적 | 우선순위 |
| ----- | ----- | ----- |
| ① 모델명 검색 | 원하는 제품을 빠르게 찾기 | ⭐⭐⭐ 높음 |
| ② 찜하기 \+ 견적서 생성 | 여러 제품 선택 후 견적서 출력 | ⭐⭐⭐ 높음 |

# **① 모델명 검색 기능**

고객이 알고 있는 모델명을 검색창에 입력하면 해당 카탈로그 페이지로 즉시 이동하는 기능입니다.

## **UI 구성**

* 카탈로그 상단에 검색창 배치 (placeholder: '모델명을 입력하세요. 예: YL-2300')

* 검색 결과: 해당 탭 자동 활성화 \+ 해당 페이지로 스크롤 이동

* 검색 결과 없을 시: '해당 모델을 찾을 수 없습니다' 안내 메시지 표시

## **데이터 구조**

모델명과 페이지 번호를 아래 형태로 코드에 미리 정의합니다. (페이지 번호는 추후 전달 예정)

const searchIndex \= \[

  { model: "YL-2300", tab: "제품A", page: 23 },  
  { model: "YL-3500", tab: "제품B", page: 45 },  
\];

## **사용자 흐름**

| 1단계  검색창에 모델명 입력 (예: YL-2300) |
| :---- |
| **2단계**  searchIndex에서 일치하는 모델 탐색 |
| **3단계**  해당 탭(제품A) 자동 클릭 활성화 |
| **4단계**  해당 페이지(23p)로 자동 스크롤 이동 |
| **5단계**  해당 페이지 테두리 하이라이트 표시 (선택사항) |

## **기술 요구사항**

* 검색은 완전 일치 \+ 부분 일치 모두 지원 (예: 'YL-23' 입력 시 YL-2300 포함 결과 표시)

* 대소문자 구분 없이 검색

* 모바일에서도 동일하게 작동

# **② 제품 찜하기 \+ 견적서 생성**

고객이 카탈로그를 보면서 원하는 제품을 찜하고, 찜한 목록을 견적서 형태로 출력하거나 카카오톡으로 전송하는 기능입니다.

## **UI 구성**

**찜하기 버튼**

* 각 PDF 페이지 우측 상단에 \[★ 찜하기\] 버튼 오버레이

* 찜 완료 시 버튼 색상 변경 (빈 별 → 채워진 별)

* 하단 고정 바에 찜한 제품 수 실시간 표시 (예: '찜한 제품 3개 | 견적서 보기')

**찜 목록 패널**

* 찜한 제품 목록 표시: 제품명, 페이지 번호, 수량 입력칸

* 개별 삭제 가능

* '전체 삭제' 버튼

**견적서 출력**

* \[견적서 PDF 출력\] 버튼 클릭 시 아래 형태의 견적서 자동 생성 및 다운로드

* \[카카오톡으로 보내기\] 버튼 클릭 시 찜 목록 텍스트가 자동 입력된 채로 채널 연결

## **견적서 양식**

| 항목 | 내용 |
| ----- | ----- |
| 상단 로고 | Zart (대산우드랜드) 로고 |
| 고객 정보 | 현장명, 담당자명, 연락처 (출력 전 입력 팝업) |
| 제품 목록 | 제품명 | 모델명 | 카탈로그 페이지 | 수량 | 비고 |
| 하단 문의처 | 카카오톡 채널 링크, 전화번호 |
| 출력일자 | 자동 삽입 |

## **카카오톡 자동 메시지 형식**

안녕하세요, 아래 제품 견적 요청드립니다.

   
■ 현장명: \[자동입력\]  
■ 요청 제품 목록:  
  1\. YL-2300 / 수량: 50ea (카탈로그 23p)  
  2\. YL-3500 / 수량: 30ea (카탈로그 45p)  
■ 연락처: 

## **기술 요구사항**

* 찜 목록은 브라우저 세션 동안 유지 (페이지 새로고침 시에도 유지: localStorage 사용)

* 견적서 PDF 생성: jsPDF 또는 html2canvas 라이브러리 활용

* 카카오톡 연결: 기존 채널 링크에 텍스트 파라미터 추가

* 모바일에서도 동일하게 작동
