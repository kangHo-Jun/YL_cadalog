# **1\. 프로젝트 개요**

A사 PDF 카탈로그에서 취급 제품 페이지만 필터링하여 대리점 전용 웹 카탈로그를 구축합니다.

기존 PDF를 페이지 단위로 재조합하여 활용하므로 제작 비용을 최소화하고, PDF만 교체하면 자동으로 업데이트됩니다.

* 카테고리 탭: 4개 (제품A, 제품B, 제품C, 제품D)

* 견적/문의: 카카오톡 채널 연결

* PDF 뷰어: PDF.js 라이브러리 사용 (카페24 FTP 호스팅)

* 반응형: PC 및 모바일 모두 지원

# **2\. 사용자 흐름**

방문자가 카탈로그를 탐색하는 전체 경로입니다.

* 쇼핑몰 방문

* 상단 GNB 메뉴 '제품 카탈로그' 클릭

* 카페24 내 전용 페이지로 이동 (헤더/푸터 유지)

* 카테고리 탭 클릭 → 해당 제품 PDF 페이지 표시

* 견적/문의 버튼 클릭 → 카카오톡 채널 이동

# **5\. 기술 스택**

* PDF.js — PDF 렌더링 라이브러리 (Mozilla 오픈소스)

* 카페24 FTP — PDF 및 라이브러리 파일 호스팅

* HTML / CSS / JavaScript — 탭 UI 및 뷰어 구현

* 카카오톡 채널 — 견적/문의 연결

# **6\. 비고**

* PDF 파일은 카페24 FTP에 업로드 후 URL 경로를 코드에 입력합니다.

* 페이지 매핑 설정 파일(catalog 객체)만 수정하면 카테고리 구성을 변경할 수 있습니다.

* A사 PDF가 업데이트될 경우 FTP에 동일한 파일명으로 재업로드하면 자동 반영됩니다.

* 카카오톡 채널 URL은 채널 관리자 페이지에서 확인 가능합니다.
