📋 대리점 웹 카탈로그 기획 문서

작성일: 2026-02-27  
프로젝트명: 대리점 전용 웹 카탈로그  
플랫폼: 카페24 쇼핑몰 내 구현

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
1\. 프로젝트 개요  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
목적: A사 PDF 카탈로그에서 취급 제품 페이지만 필터링하여 대리점 전용 웹 카탈로그 구축  
핵심 아이디어: PDF 새로 제작 X → 기존 A사 PDF를 페이지 단위로 재조합하여 활용  
플랫폼: 카페24 쇼핑몰 내 특정 페이지에 섹션으로 삽입  
타겟: 대리점 고객 (B2B)  
기대 효과: 제작 비용 최소화, PDF만 교체하면 자동 반영

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
2\. 사용자 흐름  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
쇼핑몰 방문  
→ 상단 GNB 메뉴 "제품 카탈로그" 클릭  
→ 카페24 내 별도 페이지 이동 (헤더/디자인 유지)  
→ 카테고리 탭 클릭 → 해당 제품 PDF 페이지 표시  
→ 견적/문의 버튼 클릭

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
3\. 페이지 구조  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
\[카페24 기본 헤더 \- 변경 없음\]  
GNB: 홈 | 상품 | 제품카탈로그★ | 고객센터

── 카탈로그 섹션 ──  
\[ 제품A \] \[ 제품B \] \[ 제품C \] \[ 제품D \] …  ← 카테고리 탭  
─────────────────────────────  
PDF 뷰어 영역  
\- 선택한 카테고리 페이지만 렌더링  
\- 비연속 페이지도 자연스럽게 이어서 표시  
\- 위아래 스크롤로 탐색

\[ 견적 / 문의 버튼 \]  
── 섹션 끝 ──

\[카페24 기본 푸터 \- 변경 없음\]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
4\. 기능 상세  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
GNB 메뉴: 카페24 관리자 디자인 편집에서 "제품 카탈로그" 추가

카테고리 탭:  
\- 4\~7개 제품 탭 구성  
\- 비연속 페이지 지원 (예: 3,5,8페이지 → 하나의 스크롤로 연결)  
\- 선택된 탭 활성화 표시

PDF 뷰어:  
\- PDF.js 라이브러리 사용  
\- 카페24 FTP에 PDF 파일 업로드  
\- PC/모바일 반응형

견적/문의 버튼:  
\- 연결 방식 미정 (카카오톡 / 전화 / 이메일 / 문의폼)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
5\. 카테고리별 페이지 매핑 (작성 필요)  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
제품A | 페이지: 미정  
제품B | 페이지: 미정  
제품C | 페이지: 미정  
제품D | 페이지: 미정  
제품E | 페이지: 미정

코드 설정 예시:  
const catalog \= {  
  "제품A": \[1, 2, 3, 4, 5\],  
  "제품B": \[4, 5\],  
  "제품C": \[3, 5, 8\],  
};  
※ 이 파일만 수정하면 페이지 구성 변경 완료

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
6\. 개발 순서  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  
1단계: 카탈로그 전용 페이지 생성 (카페24 관리자)  
2단계: GNB 메뉴에 "제품 카탈로그" 추가  
3단계: PDF.js 파일 FTP 업로드  
4단계: A사 PDF 파일 FTP 업로드  
5단계: 카테고리 탭 UI 제작  
6단계: PDF 렌더링 엔진 연결  
7단계: 카테고리별 페이지 매핑 설정 ★페이지 번호 필요  
8단계: 견적/문의 버튼 연결 ★연결 방식 결정 필요  
9단계: 모바일 반응형 조정  
10단계: 최종 테스트
